== Description ==
Africbase Audio Player For Wordpress ,makes It Easy To Add Audio With A Download Button Using.

This Plugin Make It very easy to add audio player with a download button with just one link, and helps boost your music ranking on Google.

Ordered list:

1.Amazing Audio Player
2.Download Button 
3.Shortcode
4.Seo Player



Link to [Africbase](http://africbase.com/ "Your favorite software") 

To Use Shortcode In Post Editor
[africbase name="Song Name" artist="Artist Name" url="song url.mp3"]

To Use Second Shortcode In Post Editor
[abplayer name="Song Name" artist="Artist Name" url="song url.mp3"]

== Installation ==
1. Upload "africbase-player" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==
= How to Use The Player =
Add With Shortcode [africbase name="Song Name" artist="Artist Name" url="link To The Song.mp3 Link"].

= Change Player Watermark  =
Add .africbase::after { background-image: url("watermark url/link")!important; } To Additional Css
